Description

IdolObject is the core object of Idol; a singleton which controls all Idol's main functions.

Details

Idol

typedef struct _IdolObject Idol;

The Idol object is a handy synonym for IdolObject, and the two can be used interchangably.

IdolObject

typedef struct _IdolObject IdolObject;

All the fields in the IdolObject structure are private and should never be accessed directly.

Property Details

The "current-mrl" property

  "current-mrl"              gchar*                : Read

The MRL of the current stream.

Default value: NULL

The "current-time" property

  "current-time"             gint64                : Read

The player's position (time) in the current stream, in milliseconds.

Default value: 0

The "fullscreen" property

  "fullscreen"               gboolean              : Read

If TRUE, Idol is in fullscreen mode.

Default value: FALSE

The "playing" property

  "playing"                  gboolean              : Read

If TRUE, Idol is playing an audio or video file.

Default value: FALSE
